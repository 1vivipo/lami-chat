# Lami Chat

简洁、纯粹的聊天应用

## 功能特性

- 📱 简洁的聊天界面
- 👤 个人资料管理
- 👥 好友系统
- 🎨 主题切换（浅色/深色/跟随系统）
- 🔔 消息通知
- 🔒 安全认证

## 技术栈

- **前端**: React Native + Expo
- **后端**: Supabase
- **状态管理**: Zustand
- **导航**: Expo Router

## 开发

```bash
# 安装依赖
bun install

# 启动开发服务器
bun start

# Android
bun run android

# iOS
bun run ios
```

## 构建

```bash
# 构建 APK
eas build --platform android --profile preview
```

## 下载
